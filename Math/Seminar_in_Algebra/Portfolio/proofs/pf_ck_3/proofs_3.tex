    \item (\textbf{Compare and Contrast})  
    
    Show that for \(p\) a prime, the polynomial \(x^{p} + a\) in \(\Z_{p} [x]\) is not irreducible for any \(a \in \Z_{p}\).
    \pf{
        Since \(p\) is prime, the degree of our polynomial is \(p \ge 2\). We know that a polynomial of degree 2 or higher is not irreducible if it has a zero in the field. So, we plan to find a zero. Recall that the Corollary to the Little Theorem of Fermat tells us that \(x^{p} \equiv x \pmod{p}\) for any \(x \in \Z\). Thus, evaluating the polynomial at \(x = -a\) gives
        \[
            (-a)^{p} + a \equiv -a + a \equiv 0 \pmod{p}.
        \]
        Therefore, we have shown that \(-a\) is a zero, meaning the polynomial can be factored and is not irreducible.
    }

    \item 

    Show that a factor ring of a field is either the trivial (zero) ring of one element or is isomorphic to the field. 
    \pf{
        Let \(F\) be a field with ideal \(N\). Since \(F\) is a field and \(N\) is an ideal, either \(N = F\) or \(N = \{0\}\). If \(N = F\), then we know from Exercise 18, that for all \(a \in F\), \(\varphi(a) = 0'\). So, the factor ring \(F/F\) consists of only one element (the identity element), it is the trivial ring. Conversely, if \(N = \{0\}\), define a mapping \(\varphi : F \to F/\{0\}\) by \(x + \{0\}\) for \(x \in F\). This mapping is surjective by definition. It is also injective by Exercise 18. Thus, \(\varphi\) is a bijective homomorphism, meaning it is an isomorphism. Therefore, \(F/\{0\} \cong F\).
    }

    \item 

    Show that each homomorphism from a field to a ring is either one to one or maps everything to 0.
    \pf{
        Let \(\varphi : F \to R\) be a ring homomorphism with kernel \(N\). Since \(F\) is a field and \(N\) is an ideal, we know either \(N = F\) or \(N = \{0\}\) because a field can have no proper nontrivial ideals. If \(N = F\), then every element in \(F\) is in \(N\). So, for all \(a \in F\), \(\varphi(a) = 0'\) for \(0' \in R\). If \(N = \{0\}\), then \(\varphi\) is one-to-one.
    }
