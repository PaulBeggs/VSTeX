\section*{Cover Sheet}

I enjoyed this homework overall, even if some parts where very confusing at times. I'm also faced with a conundrum between saying ``Tower Theorem'' and Theorem 31.4. That is the former sounds awesome, and you immediately understand why it has that name, but the latter is Theorem \(\pi\). That said, Exercise 1 was Exercise 4 on Section 30, so that was straightforward. I felt like I did Exercise 5 correctly, but ensuring all the edge cases (e.g., making sure the polynomial is irreducible, or that at least \(\beta \notin F(\alpha)\), or the elements in the bases are right given \(\alpha\)) was accounted for was a pain. At first, for Exercise 11, I set \(\alpha = \sqrt{2} + \sqrt{3}\) and solved it like in previous homework, but I looked in the back of the book to check my answer, and they only had basis \(\{1, \sqrt{2}\}\), implying I did something wrong. I went back to my notes and saw the example that we did in class that showed \(\Q(\sqrt{2} + \sqrt{3}) = \Q(\sqrt{2},\sqrt{3})\), and was able to use that fact in the Exercise. From doing these Exercises, I started to build a process of what needs to happen to get the degree of the field. That is, start with \(E(\beta + \alpha)\) or \(E(\beta_{1}, \beta_{2})\) over \(F(\alpha)\), and turn \(E(\beta + \alpha)\) into \(E(\beta, \alpha)\) or one of the \(\beta\)s equal to \(\alpha\). Then, show \(\beta \notin F(\alpha)\) because of degree, and show it is the root an irreducible polynomial via 2nd or 3rd degree Theorem or Eisenstein's. Wrap things up by finding \([E : F]\) (or \([K : F]\) if multiple finite extensions), and build a basis using Theorem 31.4. I'm not entirely sure if this is the most efficient or correct way to do things, but I think it works. In Exercise 24, I started by getting \(\sqrt{3}\) in \(\Q(\sqrt[3]{2})\) to make \(\Q(\sqrt{3})\) a field extension. Then, I could build a tower and point out that the two sides of the equation do not equal each other, so it must be that \(\sqrt{3} \notin \Q(\sqrt[3]{2})\). The next Exercise was like a generalization of the previous one, but just had slightly different wording.

\section*{Section 31}

\textbf{Computations}

In Exercises 1, 5, 11, and 13, find the degree and a basis for the given field extension. Be prepared to justify your answers. 

\begin{enumerate}
	\item \(\Q(\sqrt{2})\) over \(\Q\).
	\sol{
		For \(\alpha = \sqrt{2}\), we find \(\irr{\sqrt{2}, \Q} = x^{2} - 2\), which is of degree 2, and is irreducible by Eisenstein's criterion for \(p = 2\). Hence, \(\Q(\sqrt{2})\) is a 2-dimensional vector space over \(\Q\) with basis \(\{1, \sqrt{2}\}\) by Theorem 30.23. 
	}

	\setcounter{enumi}{4}
	\item \(\Q(\sqrt{2}, \sqrt[3]{2})\) over \(\Q\)
	\sol{
		Following from Example 31.10, we start by showing that \(\sqrt{2} \notin \Q(\sqrt[3]{2})\) because \(\deg(\sqrt{2}, \Q) = 2\) and 2 is not a divisor of \(3 = \deg(\sqrt[3]{2}, \Q)\). Therefore, we have \(\Q(\sqrt{2}, \sqrt[3]{2})\) over \(\Q(\sqrt[3]{2})\) with \(f(x) = x^{2} - 2\). Thus, 
		\[
			[\Q(\sqrt{2}, \sqrt[3]{2}) : \Q(\sqrt[3]{2})] = \deg(f(x)) = 2.
		\]
		Then, we find \(\Q(\sqrt[3]{2})\) over \(\Q\) has degree \([\Q(\sqrt[3]{2}) : \Q] = 3\) for \(g(x) = x^{3} - 2\). Then, we use Theorem 30.23 to produce bases \(\{1, \sqrt{2}\}\) for \(\Q(\sqrt{2}, \sqrt[3]{2})\) over \(\Q(\sqrt[3]{2})\), and \(\{1, \sqrt[3]{2}, (\sqrt[3]{2})^{2}\}\) for \(\Q(\sqrt[3]{2})\) over \(\Q\). By Theorem 31.4, we find 
		\[
			[\Q(\sqrt{2}, \sqrt[3]{2}) : \Q] = [\Q(\sqrt{2}, \sqrt[3]{2}) : \Q(\sqrt[3]{2})][\Q(\sqrt[3]{2}) : \Q] = 6,
		\] 
		and we get a basis (from the proof of Theorem 31.4)
		\[
			\{1, \sqrt{2}, \sqrt[3]{2}, \sqrt{2}\sqrt[3]{2}, (\sqrt[3]{2})^{2}, \sqrt{2}(\sqrt[3]{2})^{2}\}
		\]
		for \(\Q(\sqrt{2}, \sqrt[3]{2})\) over \(\Q\). Finally, we note that \(f(x)\) is irreducible over \(\Q(\sqrt[3]{2})\) by Theorem 23.10, and \(g(x)\) is irreducible over \(\Q\) by Eisenstein's criterion for \(p = 2\).
	}

	\setcounter{enumi}{10}

	\item \(\Q(\sqrt{2} + \sqrt{3})\) over \(\Q(\sqrt{3})\)
	\sol{
		The example in our notes shows that \(\Q(\sqrt{2} + \sqrt{3}) = \Q(\sqrt{2},\sqrt{3})\) (and Exercise 28 states this is true for all \(\sqrt{a} + \sqrt{b} \ne 0\) for \(a,b \in \Q\)). Then, for \(\alpha = \sqrt{2}\) we have \(\irr{\sqrt{2}, \Q(\sqrt{3})} = x^{2} - 2\), which is irreducible over \(\Q(\sqrt{3})\) by Theorem 23.10. Thus, we find \([\Q(\sqrt{2},\sqrt{3}) : \Q(\sqrt{3})] = 2\) with a basis \(\{1, \sqrt{2}\}\).
	}


\newpage
	\setcounter{enumi}{12}
	\item \(\Q(\sqrt{2}, \sqrt{6} + \sqrt{10})\) over \(\Q(\sqrt{3} + \sqrt{5})\)
	\sol{
		We see that \(\sqrt{6} + \sqrt{10} = \sqrt{2}(\sqrt{3} + \sqrt{5})\), so 
		\begin{equation}
			\Q(\sqrt{2}, \sqrt{6} + \sqrt{10}) = \Q(\sqrt{2}, \sqrt{2}(\sqrt{3} + \sqrt{5}))
		\end{equation}
		because we can factor inside the field \(\Q(\sqrt{2}, \sqrt{6} + \sqrt{10})\). We want to show that (1) is equal to \(\Q(\sqrt{2}, \sqrt{3} + \sqrt{5})\). Notice that the product of elements \(\sqrt{2}, \sqrt{3} + \sqrt{5} \in \Q(\sqrt{2}, \sqrt{3} + \sqrt{5})\), is also in \(\Q(\sqrt{2}, \sqrt{2}(\sqrt{3} + \sqrt{5}))\). Then, for elements \(\sqrt{2}, \sqrt{2}(\sqrt{3} + \sqrt{5}) \in \Q(\sqrt{2}, \sqrt{2}(\sqrt{3} + \sqrt{5}))\), we compute 
		\[
			\frac{\sqrt{2}(\sqrt{3} + \sqrt{5})}{\sqrt{2}} = (\sqrt{3} + \sqrt{5}),
		\]
		to see that \(\sqrt{2}, \sqrt{2}(\sqrt{3} + \sqrt{5}) \in \Q(\sqrt{2}, \sqrt{3} + \sqrt{5})\). Thus, 
		\[
			\Q(\sqrt{2}, \sqrt{2}(\sqrt{3} + \sqrt{5})) = \Q(\sqrt{2}, \sqrt{3} + \sqrt{5}).
		\]
		Thus, \([\Q(\sqrt{2}, \sqrt{3} + \sqrt{5}) : \Q(\sqrt{3} + \sqrt{5})] = 2\) for \(f(x) = x^{2} - 2\) with basis \(\{1, \sqrt{2}\}\).
	}
\end{enumerate}

\textbf{Theory}

\begin{enumerate}
	\setcounter{enumi}{23}
	\item Prove that \(x^{2} - 3\) is irreducible over \(\Q(\sqrt[3]{2})\).
	\pf{
		Suppose, for the sake of a contradiction, that \(x^{2} - 3\) is not irreducible over \(\Q(\sqrt[3]{2})\). That is, \(\sqrt{3} \in \Q(\sqrt[3]{2})\), so \(\Q(\sqrt{3})\) is a field extension of \(\Q(\sqrt[3]{2})\). Then, by Theorem 31.4, \([\Q(\sqrt[3]{2}) : \Q] = [\Q(\sqrt[3]{2}) : \Q(\sqrt{3})] \cdot [\Q(\sqrt{3}) : \Q]\). We know \([\Q(\sqrt[3]{2}) : \Q] = 3\) and \([\Q(\sqrt{3}) : \Q] = 2\). This would mean \(3 = [\Q(\sqrt[3]{2}) : \Q(\sqrt{3})] \cdot 2\), implying 2 divides 3, which is not true. Therefore, \(\sqrt{3} \notin \Q(\sqrt[3]{2})\), and by Theorem 23.10, the polynomial \(x^{2} - 3\) is irreducible.
	}

	\setcounter{enumi}{28}

	\item Let \(E\) be a finite extension of a field \(F\), and let \(p(x) \in F[x]\) be irreducible over \(F\) and have degree that is not a divisor of \([E : F]\). Show that \(p(x)\) has no zeros in \(E\).
	\pf{
		Suppose \(\deg(p(x))\) is not a divisor of \([E : F]\), and for the sake of a contradiction, that \(p(x)\) does have a zero in \(E\). That is, there exists an \(\alpha \in E\) such that \(E\) forms a finite extension field of \(F(\alpha)\). This allows us to use Theorem 31.4:
		\[
			[E : F] = [E : F(\alpha)][F(\alpha) : F],
		\]
		where \([F(\alpha) : F] = \deg(p(x))\). Since \([E : F(\alpha)]\) is finite, it must be an integer. Therefore, \(\deg(p(x))\) must divide \([E : F]\). However, this leads to a contradiction because we assumed \(\deg(p(x))\) is not a divisor of \([E : F]\). Therefore, \(p(x)\) has no zeros in \(E\).
	}

\end{enumerate}
