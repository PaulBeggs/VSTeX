\section*{Section 20}

In Exercises 11 through 14, describe all solutions of the given congruence, as we did in Examples 20.14 and 20.15.

\begin{enumerate}
    \setcounter{enumi}{10}
    \item \(2x \equiv 6 \pmod{4}\)
          \sol{
              (\textit{Note:} Even though this first problem is fairly straightforward, I want to go into excruciating detail so that the process is properly established for Exercises 12-14.) First, we find \(d = \gcd(2,4) = 2\), which does divide 6. The comment to Corollary 20.13 tells us that the solutions to this congruence are all elements of the residue classes
              \[
                  (s + km_{1}) + (m\Z) \quad \text{where} \quad m_{1} = m / d
              \]
              and \(k\) runs through all integers from 0 to \(d - 1\). Additionally, we find \(s\) by solving for \(x\) in the equation \(a_{1}x = b_{1}\) in \(\Z_{m}\), where \(a_{1} = a / d\) and \(b_{1} = b / d\). Now, we can apply this to our problem. We get
              \[
                  d = 2, \quad m_{1} = 2, \quad k \in \{0,1\}, \quad a_{1} = 1, \quad \text{and} \quad b_{1} = 3.
              \]
              These values give us the equation \(x = 3\) in \(\Z_{2}\). Solving, we see that \(x = 1\), giving the solution \(s = 1\). Thus, our residue classes are
              \[
                  (1 + (0)2) + 4\Z = 1 + 4\Z,
              \]
              and
              \[
                  (1 + 1(2)) + 4\Z = 3 + 4\Z.
              \]
          }
    \item \(22x \equiv 5 \pmod{15}\)
          \sol{
              We calculate \(d = \gcd(22, 15)\) and find \(d = 1\). Thus, we know there is only one solution, which we find to be \(x = 5\) by repeatedly trying different values. Hence, this gives us \(5 + 15\Z\) as our set of solutions.
          }
    \item \(36x \equiv 15 \pmod{24}\)
          \sol{
              We calculate \(d = \gcd(36, 24) = 12\), which we note does not divide 15. Hence, this congruence does not have any solutions.
          }
\newpage
    \item \(45x \equiv 15 \pmod{24}\)
          \sol{
              We calculate \(d = \gcd(45,24) = 3\), which does divide 15. Thus, we can assemble our components:
              \[
                  d = 3, \quad m_{1} = 8, \quad k \in \{0,1,2\}, \quad a_{1} = 15, \quad \text{and} \quad b_{1} = 5.
              \]
              With these components, we assemble the equation \(15x = 5\) in \(\Z_{8}\). (Note we can further reduce this to \(7x = 5\) because we are in \(\Z_{8}\).) We find \(x = 3\) by trying different combinations. This gives us the solution \(s = 3\). Thus, our residue classes are
              \[
                  3 + 24\Z, \quad 11 + 24\Z, \quad \text{and} \quad 19 + 24\Z.
              \]
          }
\end{enumerate}

              \definecolor{first}{HTML}{65d3f5}

\begin{enumerate}

    \item Let \(R\) be a nonzero commutative ring, and let \(T\) be a nonempty subset of \(R\) closed under multiplication and containing neither 0 nor divisors of 0. Starting with \(R \times T\) and otherwise exactly following the construction in this section, we can show that the ring \(R\) can be enlarged to a partial ring of quotients \(Q(R, T)\). Think about this for 15 minutes or so; look back over the construction and see why things still work. In particular, show the following:
          \begin{enumerate}
              \item \(Q(R, T)\) has unity even if \(R\) does not.
                    \pf{
                        To show that \(Q(R,T)\) has unity, consider an element \(t \in T\), which exists because \(T\) is nonempty. Now, let \(a \in R\) and \(b \in T\) so that \([(a,b)] \in Q(R,T)\). This gives us the computation
                        \[
                            [(t,t)][(a,b)] = [(ta, tb)],
                        \]
                        which is allowed by Lemma 21.3. Then, by Lemma 21.2, we know that \((ta, tb) \sim (a,b)\) because \(tab = tba\), which holds since the elements \(a\) and \(b\) commute in \(R\). (A symmetrical argument holds that shows \((at, bt) \sim (a,b)\).) Thus, \([(t,t)]\) is the unity of \(Q(R,T)\).
                    }
              \item In \(Q(R, T)\), every nonzero element of \(T\) is a unit. (\textit{Note:} You must show that for non-zero \(t\) in \(T\), \([(tt, t)]\) is a unit in \(Q(R,T)\). You should explain why \(t\) is represented by the equivalence class \([(tt, t)]\) as opposed to \([(t, 1)]\) as in Lemma 21.4.)
                    \pf{
                        To show that every nonzero element of \(T\) is a unit, consider the equivalence class \([(tt,t)]\). We need to show that for some \([(a,b)] \in Q(R,T)\), that \([(tt,t)][(a,b)] = [(t,t)]\). We can choose \(a = t\) and \(b = tt\) to be this equivalence class because \(t \in R\) (as \(T \subseteq R\)), and multiplication is closed in \(T\), so the product of \(t\cdot t\) is contained. Thus, consider the computation
                        \[
                            [(tt,t)][(t,tt)] = [(ttt,ttt)].
                        \]
                        Then, we know \([ttt,ttt]\) is equivalent to \([t,t]\) because \(tttt = tttt\). Thus, we have shown that every nonzero element of \(T\) is a unit. Note that we choose \([(tt,t)]\) instead of \([(t,1)]\), as we cannot assume that \(1 \in T\) (which would then imply \(1 \in R\), which we also don't know). Then, we find \([(tt,t)]\) by multiplying \(t \in R\) by \(t\), which is contained because \(R\) is a nonzero commutative ring. 
                    }
          \end{enumerate}
